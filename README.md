# 恶搞程序

恶搞程序是一个基于 Python 的恶搞程序，它可以让你在屏幕上看到各种恶搞的场景。

## 使用方法

1. 准备 Python 3.0 及以上的版本环境。

2. 下载或克隆本仓库到本地。

3. 安装依赖库：

   ```
   pip install -r requirements.txt
   ```

4. 运行以下命令启动程序：

   ```
   python main.py
   ```
